## File Server (AWS S3)

In this project, I created a file server for a basic video sharing site called Tubely. The backend is implemented in Go and the front end is implemented with Angular. The backend was provided in a partially-complete state, and the front end was provided fully completed; this project mainly focused on managing file uploads, storing files in S3 buckets, and implementing role based privileges in AWS.

### Skills Gained
- AWS S3 Usage
- AWS CLI Usage
- FFMPEG
- AWS Role/Group Definition
- AWS Authentication/IAM
- CloudFront
