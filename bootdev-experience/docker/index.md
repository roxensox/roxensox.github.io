## Docker

In this class, I learned how to run and manage docker containers through the command line, as well as configure and create new containers. The containers created included a load balancer, a simple Go web server, and a Python book-bot that performs basic analysis on text files.

### Skills Gained
- Docker CLI
- Docker Hub
- Docker Network Controls
- Docker File Creation
- Docker Environment Control
